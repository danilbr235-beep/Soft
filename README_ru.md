# Soft Goals Manager (Русская версия)

Soft Goals Manager — это персональный менеджер целей с поддержкой командной строки, REST API и современного веб‑интерфейса.

## Возможности

- **CLI (командная строка):** позволяет управлять целями через терминал. Вы можете добавлять цели с описанием, указать дату выполнения, категорию и приоритет. В списке целей доступны фильтры по категории, приоритету, просроченным задачам, задачам до указанного количества дней, а также сортировка. Поддерживается редактирование целей, отметка выполнения/отмены и удаление. Данные сохраняются в файле `goals.json`.
- **REST API сервер:** построен на Express.js и хранит данные в `goals.json`. Предоставляет эндпойнты для получения списка целей, создания, обновления и удаления. Каждая цель содержит поля `text`, `dueDate`, `category`, `priority` (по умолчанию "medium") и `completed`.
- **Веб‑интерфейс:** адаптивный интерфейс на HTML/CSS/JavaScript. Поддерживает тёмную тему и переключатель тем, поиск, фильтры по категории/приоритету, просроченным задачам и задачам до выбранного срока, сортировку, отображение статистики (всего, завершено, просрочено, до 7 дней). Позволяет редактировать, завершать и удалять задачи прямо в браузере. Пользователь получает уведомления о действиях.

## Требования

- Node.js версии 14 и выше
- npm (идёт в комплекте с Node.js)

## Установка

Клонируйте репозиторий и установите зависимости:

```bash
git clone https://github.com/danilbr235-beep/Soft.git
cd Soft
npm install
```

## Запуск веб‑приложения

Запустите сервер Express (обслуживает API и статические файлы в `/public`):

```bash
npm start
```

По умолчанию сервер работает на порту 3000. Откройте `http://localhost:3000` в браузере, чтобы воспользоваться приложением.

## Использование CLI

Используйте npx или node для запуска CLI:

```bash
# Добавить цель с датой, категорией и приоритетом
npx soft-cli add \"Прочитать книгу\" --due=2026-05-01 --category=образование --priority=high

# Показать список целей (с фильтрами по категории, приоритету, просроченным задачам, сроку)
npx soft-cli list --category=образование --priority=high --due-within=7 --overdue

# Изменить цель: текст, дату, категорию или приоритет
npx soft-cli edit <id> --text=\"Прочитать две книги\" --due=2026-05-10 --category=образование --priority=medium

# Отметить цель выполненной или отменить выполнение
npx soft-cli complete <id>

# Удалить цель
npx soft-cli delete <id>

# Посмотреть перечень команд
npx soft-cli help
```

## Структура проекта

- **app_priority.js** — сервер Express с REST API и поддержкой приоритета.
- **index_priority.js** — CLI‑скрипт для управления целями.
- **public/index.html** — веб‑интерфейс.
- **public/styles_v5.css** — стили для светлой/тёмной тем, компоновки, бейджей и уведомлений.
- **public/new_script_v7.js** — клиентский JavaScript с поиском, фильтрами, сортировкой, статистикой, тёмной темой и бейджами приоритета.
- **goals.json** — файл JSON для хранения целей.
