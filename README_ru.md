# Soft Goals Manager (Русская версия)

Soft Goals Manager — это персональный менеджер целей с поддержкой командной строки, REST API (Express) и простого веб‑интерфейса.

## Требования

- Node.js версии 14 и выше
- npm

## Установка

```
git clone https://github.com/danilbr235-beep/Soft.git
cd Soft
npm install
```

## Запуск веб‑приложения

```
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`.

## Использование CLI

```
# Добавить цель
npx soft-cli add "Прочитать книгу" --due=2026-05-01 --category=образование

# Вывести список целей
npx soft-cli list --category=образование --due-within=7

# Изменить цель
npx soft-cli edit <id> --text="Прочитать две книги" --due=2026-05-10 --category=образование

# Отметить выполненной или удалить
npx soft-cli complete <id>
npx soft-cli delete <id>
```

## Описание проекта

Проект состоит из трёх основных частей:

1. CLI‑утилита — позволяет управлять целями через терминал (добавлять, перечислять, завершать, удалять, редактировать и фильтровать).
2. REST API на Express — хранит цели в файле JSON, предоставляет маршруты для получения, создания, обновления и удаления целей.
3. Веб‑интерфейс — интерактивная страница для работы с целями, включающая фильтры, сортировку, уведомления и статистику.
