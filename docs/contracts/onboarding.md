# Onboarding Contract

Onboarding collects the minimum baseline needed to produce a useful Today screen.

Required steps:

1. Welcome and trust framing
2. Privacy and expectations
3. Primary goal
4. Baseline snapshot
5. App mode
6. First Today screen

Required output:

- primary goal
- secondary goals
- Simple or Pro mode
- baseline sleep, energy, confidence, libido, stress
- conservative guidance flag
- privacy preferences
- content language preference
- starter program recommendation
- onboarding completion timestamp

The flow must allow sensitive questions to be skipped.

## Implemented Fast Path

The current scaffold implements:

- privacy welcome
- primary goal
- baseline preset
- Simple/Pro mode
- conservative guidance toggle
- generated Today screen
